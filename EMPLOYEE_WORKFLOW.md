# Employee Workflow

The Employee is the standard end-user who interacts with the system to book resources and manage their assigned assets.

## Responsibilities & Daily Tasks

1. **Daily Usage**
   - Log into the dashboard to view all assets currently assigned to them.
2. **Resource Booking**
   - Use the calendar view to search for and book shared resources for specific time slots (e.g., projectors, vehicles).
   - The system automatically prevents overlapping bookings.
3. **Issue Reporting**
   - If an assigned asset breaks or requires servicing, raise a Maintenance Request.
   - Provide issue details, set priority, and attach photos.
4. **Returns & Transfers**
   - Initiate return requests when an asset is no longer needed.
   - Initiate transfer requests if an asset needs to be handed over to another employee or department.
