# Admin Workflow

The Admin is responsible for managing the master data that the entire AssetFlow system depends on.

## Responsibilities & Daily Tasks

1. **Onboarding & Setup**
   - Log into the system and navigate to the Organization Setup screen.
2. **Master Data Management**
   - Create and manage organizational departments.
   - Set up and manage asset categories (e.g., Electronics, Furniture).
3. **Role Assignment**
   - Review the Employee Directory.
   - Promote specific users to the `Department Head` or `Asset Manager` roles (this is the only place roles are assigned).
4. **Oversight & Auditing**
   - Create structured Audit Cycles, defining the scope (department/location) and assigning auditors.
   - Monitor organization-wide analytics on the Dashboard and Reports screens.
